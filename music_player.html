Music Player Concept
Student Project | Fall 2020


Tools Used: HTML, CSS, JavaScript

Task: Build a mobile music player application with core web technologies to understand website structure and implement a web application following web content accessibility guidelines.


The first stage consisted of implementing a two-screen music player interface showing an album and song select screen using HTML and CSS.

The four albums shown here are example albums that follow a consistent, complementary color theme.

The second stage was the implementation of the music player screens using HTML, CSS, and Javascript.

Each button triggers its corresponding action and is consistent with the music player's status, simulating playback.


Takeaways: This project was one of my first experienes working with HTML, CSS, and JavaScript and I had fun learning and gaining a new skillset. I'd like to use this knowledge in a greater capacity in the future in order to improve, problem-solve, and create more intentionally.
